Data Analytics Project

Overview

This project demonstrates a complete Data Analytics workflow, starting from data loading and cleaning to visualization and reporting. The project uses Python, SQL, and Power BI to transform raw data into meaningful business insights.

---

Dataset

- Dataset: Customer Shopping Behavior Dataset
- Format: CSV
- Contains customer demographics, purchasing behavior, product details, and sales information.

---

Tools & Technologies

- Python
  - Pandas
  - NumPy
  - Matplotlib
- Jupyter Notebook
- MySQL
- SQL Server
- Power BI
- Gamma AI (for presentation)
- Microsoft PowerPoint

---

Project Steps

1. Data Loading

- Imported the CSV dataset into Python using Pandas.
- Explored the dataset structure and data types.

2. Data Cleaning

- Removed duplicate records.
- Handled missing values.
- Renamed columns for consistency.
- Corrected data types.
- Created new calculated columns where required.

3. Exploratory Data Analysis (EDA)

- Analyzed customer demographics.
- Studied purchasing patterns.
- Examined sales trends.
- Identified key insights using summary statistics and visualizations.

4. SQL Analysis

The cleaned dataset was imported into:

- MySQL
- SQL Server

SQL queries were written to:

- Retrieve filtered records.
- Perform aggregations.
- Group and summarize data.
- Rank and sort results.
- Generate business insights.

5. Power BI Dashboard

Built an interactive dashboard including:

- KPI Cards
- Sales Analysis
- Customer Insights
- Product Performance
- Interactive Filters and Slicers
- Charts and Visualizations

6. Report Creation

Prepared a report summarizing:

- Data cleaning process
- EDA findings
- SQL analysis
- Dashboard insights
- Key business recommendations

7. Presentation

Created a professional presentation using Gamma AI and exported it to PowerPoint.

How to Run

1. Clone or download this repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook.
4. Run the notebook from start to finish.
5. Load the cleaned dataset into MySQL and SQL Server.
6. Execute the SQL scripts.
7. Open the Power BI (.pbix) file to explore the dashboard.
8. Review the report and presentation files.
